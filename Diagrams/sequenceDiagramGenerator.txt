LabellingMechanism->CacheManager:read Cache

User->Logger:Log User Creation
User-->>ConfigParser:return Users

ConfigParser-->>LabellingMechanism:return Config

LabellingMechanism->InputParser:get Input

InputParser->Dataset:create Dataset

Dataset->Instance:create Instances
Instance-->>Dataset:retrun Instances

Dataset->Label:create Labels
Label-->>Dataset:retrun Labels

Dataset-->>InputParser:retrun Dataset

InputParser-->>LabellingMechanism:return Input

LabellingMechanism->User:Assign Labels
User->AssignedInstances:create AssignedInstance
AssignedInstances-->>User:return
User-->>LabellingMechanism:return Assigned Instances







LabellingMechanism->CacheManager:Save Cache


LabellingMechanism->ReportWriter:get Report
LabellingMechanism->OutputWriter:write Output

ReportWriter->ControllerDomain:get Metrics

ControllerDomain->Instance: get Metric Data
Instance-->>ControllerDomain: return Metric Data

ControllerDomain->Label: get Metric Data
Label-->>ControllerDomain: return Metric Data

ControllerDomain->User: get Metric Data
User-->>ControllerDomain: return Metric Data

ControllerDomain->Dataset: get Metric Data
Dataset-->>ControllerDomain: return Metric Data

ControllerDomain-->>ReportWriter:return Metrics

ReportWriter-->>LabellingMechanism:return Report
